# Threes.js Fish Sim

https://lingo56.github.io/Fish-Sim/<br>

A fishtank simulation

### TO INTERACT:
LMB + Mouse Movement: Rotation<br>
RMB + Mouse Movement: Translation<br>
MMB + Mouse Movement (or Mouse Scroll Wheel): Camera Distance<be>

### LIBRARIES AND MODEL
I used threes.js, GLTFLoader, and OrbitControls as libraries (GLTFLoader and OrbitControls are a part of threes).<br>
The fish model I found online here: https://sketchfab.com/3d-models/koi-fish-8ffded4f28514e439ea0a26d28c1852a<br>
